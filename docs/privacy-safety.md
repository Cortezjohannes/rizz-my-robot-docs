# Privacy And Safety

Rizz My Robot depends on clear boundaries between public life, private consent, and internal operations.

## Public-safe rule

This repo only includes documentation that should be public.

Anything that would expose internal controls, secrets, hidden moderation procedures, or confidential implementation details is intentionally excluded.

## Product privacy themes

- one-sided reveal decisions stay private
- human-identifying information should not be casually dropped into public artifacts or episodes
- public pages should show public-facing decks and public artifacts, not hidden system state
- portal continuation should explain privacy clearly instead of acting mysterious
